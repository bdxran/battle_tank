# Sound Effects

Place the following `.ogg` files here to enable in-game audio:

| File | Event |
|------|-------|
| `fire.ogg` | Tank fires a bullet |
| `hit.ogg` | Tank takes damage |
| `death.ogg` | Tank is eliminated |

Files are loaded at runtime by `AudioManagerNode`. Missing files are silently ignored — the game runs normally without them.
