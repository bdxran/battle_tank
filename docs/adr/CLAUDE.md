# CLAUDE.md — docs/adr/

Ce dossier contient les Architecture Decision Records (ADR) du projet.

## Format de nommage

`ADR-[NNN]-[titre-en-kebab-case].md`

Exemple : `ADR-001-choix-framework-websocket.md`

## Générer une ADR

```
/adr de [nom-fichier-analyse].md
```

Ou sans fichier source : `/adr` — Claude demandera le sujet.
