# CLAUDE.md — docs/analyse/

Ce dossier contient les analyses fonctionnelles du projet.

## Usage

Rédiger une analyse avant d'implémenter une mécanique complexe.
Utiliser ensuite `/std de [fichier].md` pour générer la STD correspondante.

## Format recommandé

```markdown
# Analyse — [Sujet]

## Contexte
[Pourquoi cette fonctionnalité ?]

## Comportement attendu
[Description détaillée]

## Cas limites
[Edge cases identifiés]

## Questions ouvertes
[Points à valider avant d'implémenter]
```
